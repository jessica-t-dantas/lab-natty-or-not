# Natural ou Fake Natty? Como Vencer na Era das IAs Generativas

### Template

```markdown
# 🌌 Projeto: Extremamente Aesthetic

## 📒 Descrição
Um projeto que utiliza IA para gerar dicas de saúde com elementos visuais, como guias de alongamentos, exemplos de refeições saudáveis e técnicas de respiração. O objetivo é criar uma experiência rica e intuitiva, onde o usuário pode visualizar as dicas em vez de apenas lê-las.

## 🤖 Tecnologias Utilizadas
Chat GPT para geração do texto
Leonardo AI para geração das imagens

## 🧐 Processo de Criação

Planejar categorias visuais: Dividir as dicas em categorias (ex.: alimentação saudável, alongamentos, respiração guiada) e criar animações e tutoriais visuais para cada uma.
Desenvolver ilustrações detalhadas: Gerar imagens que representem pratos balanceados, posições de yoga, exercícios de respiração com animações de ar subindo e descendo, etc.
Personalização e interatividade: A IA ajusta os tutoriais visuais com base nas respostas do usuário, como nível de preparo físico, objetivos de saúde ou tempo disponível, e permite interação para que o usuário explore cada elemento visual.
```
